# Muneeb Ur Tarantino — Portfolio

**Live:** [m-tarantino.github.io](https://m-tarantino.github.io)

Interactive portfolio with two interface layers:
- **Terminal Shell** (`cli.html`) — Command-line portfolio explorer
- **VNC Desktop** (`vnc.html`) — Ubuntu-style desktop with draggable windows

## Stack
- Vanilla HTML/CSS/JS (zero dependencies)
- Single `data.json` as central data source
- GitHub Pages for hosting
